# Stratégie LinkedIn — StellarWave

> **Objectif** : Générer des leads pour des missions de développement freelance
> **Approche** : 80% Inbound (contenu) / 20% Outbound (réseau)
> **Positionnement** : Dev Full-Stack Senior + Architecte Cloud

---

## 1. Principes éthiques

### ✅ Ce qu'on fait

| Principe | Application |
|----------|-------------|
| **Personnalisation** | Chaque message fait référence à un élément spécifique du profil |
| **Valeur d'abord** | On aide/conseille avant de pitcher |
| **Transparence** | Pas de faux prétexte ("J'aimerais ton avis" → pitch) |
| **Respect du silence** | Max 1 relance après 2 semaines, puis stop |
| **Cibles pertinentes** | Seulement les gens avec un besoin visible ou potentiel |

### ❌ Ce qu'on ne fait PAS

- Spam en masse (même message à 100 personnes)
- Pitch immédiat sans contexte
- Faux networking ("Échangeons !" → pitch)
- Relances insistantes (3x sans réponse)
- Connexion + pitch instantané

### 🚫 Contacts à exclure

- **Orange et filiales** (Orange Business, Orange Cyberdefense, etc.) — ne pas contacter les personnes qui y travaillent ou y ont travaillé

## 2. Stratégie Inbound (80%)

### 2.1 Contenu LinkedIn — Calendrier

| Jour | Type de contenu | Exemple |
|------|-----------------|---------|
| **Lundi** | Retour d'expérience technique | "Comment j'ai migré une app de CRA vers Next.js en 3 jours" |
| **Mercredi** | Tendance/Opinion | "Pourquoi je préfère Supabase à Firebase en 2026" |
| **Vendredi** | Projet réalisé (avec permission) | "On vient de livrer une app de facturation pour [Client]" |

**Fréquence cible** : 2-3 posts/semaine minimum

### 2.2 Formats qui fonctionnent

1. **Carousel** (5-8 slides) — Fort engagement
   - "5 erreurs à éviter en React"
   - "Architecture d'une app scalable"

2. **Post texte + image** — Bon reach
   - Screenshot de code avec explication
   - Before/After d'une interface

3. **Post personnel** — Humanise le profil
   - Leçons apprises sur un projet difficile
   - Pourquoi j'ai choisi le freelance

### 2.3 Hashtags à utiliser

```
#Freelance #DevFullStack #React #NextJS #NodeJS #TypeScript
#AWS #GCP #Cloud #Architecture #Startup #Tech #Remote
#DéveloppementWeb #SaaS #IndependantIT
```

### 2.4 Groupes/Communautés à rejoindre

- Freelance Développeurs France
- Startup & Tech France
- React & Next.js France
- AWS User Group France
- CTO, Tech Lead & Engineering Managers

## 3. Stratégie Outbound éthique (20%)

### 3.1 Demande de referrals (contacts proches)

**Cible** : Contacts 1er degré avec qui tu as déjà échangé

**Template — Referral général**
```
Salut [Prénom],

J'espère que tu vas bien !

Je développe mon activité freelance chez StellarWave (dev full-stack React/Node + Cloud AWS/GCP).

Si dans ton réseau tu croises quelqu'un qui cherche un dev senior pour un projet (app, SaaS, refonte...), je serais ravi que tu penses à moi.

Pas de pression, juste un petit signal dans ta tête !

Mon CV si besoin : https://www.stellarwave.fr/cv-fr

Merci et à bientôt,
Amar
```

**Template — Referral ciblé (ex-collègue)**
```
Salut [Prénom],

Ça fait un moment ! J'espère que [entreprise] se porte bien.

Je suis toujours en freelance et je cherche de nouvelles missions (React, Node, Cloud).

Tu aurais des contacts qui pourraient avoir besoin d'un dev senior ? Startups, scale-ups, ou même des anciens collègues qui ont bougé ?

Merci d'avance !
Amar
```

### 3.2 Activation ciblée (besoin visible)

**Signaux à surveiller** :
- 🚀 Levée de fonds annoncée
- 👔 Nouveau CTO/CPO/Head of Product
- 📢 Post "On recrute un dev"
- 🏗️ Lancement de nouveau produit
- 💼 Connexion récente avec un décideur tech

**Template — Félicitations + dispo**
```
Salut [Prénom],

Félicitations pour [la levée / le nouveau poste / le lancement] !

Je suis dev full-stack senior freelance (8+ ans, React/Node/Cloud). Si vous avez besoin de renfort tech sur vos projets, je suis disponible.

Mon profil : https://www.stellarwave.fr/cv-fr

Belle continuation !
Amar
```

**Template — Réponse à "On recrute"**
```
Salut [Prénom],

J'ai vu que vous cherchiez un dev [React/Full-Stack/etc.].

Je suis freelance senior (8+ ans), spécialisé React, Node, TypeScript + Cloud AWS/GCP. Je travaille en full remote.

Si un profil freelance peut vous intéresser (flexibilité, démarrage rapide), je serais ravi d'en discuter.

Mon CV : https://www.stellarwave.fr/cv-fr

Amar
```

### 3.3 Approche startup/scale-up

**Cible** : Fondateurs, CTO, CPO de startups early-stage (Seed/Serie A)

**Template — Proposition de valeur startup**
```
Salut [Prénom],

J'ai vu que [Startup] se développe bien — [mention d'un élément spécifique : feature, article, levée].

Je suis dev full-stack freelance senior, j'accompagne régulièrement des startups sur des sprints de développement (MVP, features critiques, refacto).

Si vous avez besoin de renfort ponctuel ou récurrent, je serais ravi d'échanger.

Portfolio : https://www.stellarwave.fr/cv-fr

Amar
```

## 4. Workflow d'automatisation (Claude Code)

### 4.1 Recherches LinkedIn prêtes

**Startups en croissance (France) :**
```
https://www.linkedin.com/search/results/people/?keywords=CTO%20startup&geoUrn=%5B%22105015875%22%5D&origin=FACETED_SEARCH
```

**CPO / Head of Product :**
```
https://www.linkedin.com/search/results/people/?keywords=%22head%20of%20product%22%20OR%20%22CPO%22%20startup&origin=GLOBAL_SEARCH_HEADER
```

**Fondateurs tech :**
```
https://www.linkedin.com/search/results/people/?keywords=founder%20tech%20startup%20france&origin=GLOBAL_SEARCH_HEADER
```

### 4.2 Workflow d'envoi

```
1. Identifier un profil avec signal (levée, recrutement, lancement)
2. Vérifier le profil (pas déjà contacté, besoin visible)
3. Personnaliser le message (élément spécifique du profil)
4. Envoyer (Message gratuit si 1er degré, InMail sinon)
5. Logger dans le tracking
6. Relancer J+14 si pas de réponse (max 1 relance)
```

## 7. Bonnes pratiques

### Volume
- **Referrals** : 5-10 messages/semaine aux contacts proches
- **Outbound ciblé** : 10-15 messages/semaine aux signaux détectés
- **Pas de spam** : Jamais plus de 30 messages/jour

### Timing
- **Meilleurs jours** : Mardi, Mercredi, Jeudi
- **Meilleurs horaires** : 8h-9h et 17h-18h
- **Éviter** : Week-end, lundi matin, vendredi après-midi

### Personnalisation obligatoire
- ✅ Prénom
- ✅ Entreprise
- ✅ Un élément spécifique (levée, post, projet)
- ✅ Proposition de valeur adaptée
